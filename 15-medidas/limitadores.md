# Limitadores de ancho y alto

Cuando establecemos un width y/o un height con una medida que es relativa es posible que no queramos que se aplique en todos los casos.

Para eso están los limitadores:

- min-width -> Indica el ancho mínimo que puede alcanzar
- max-width -> Indica el ancho máximo que puede alcanzar
- min-height -> Indica el alto mínimo que puede alcanzar
- max-height -> Indica el alto máximo que puede alcanzar
